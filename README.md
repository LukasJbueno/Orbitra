## Descripción

Orbitra es una aplicación web que permite a los usuarios explorar eventos astronómicos (eclipses, lluvias de meteoros, fases lunares, etc.) organizados por año y mes. Los usuarios pueden crear una cuenta, suscribirse a los eventos que les interesan y configurar recordatorios por correo electrónico.

## Instalación y uso local

```bash
# 1. Clonar el repositorio
git clone https://github.com/LukasJbueno/Orbitra
cd Orbitra_Fe

# 2. Instalar dependencias
npm install

# 3. Iniciar servidor de desarrollo
npm run dev

# 4. Construir para producción
npm run build
```
